Panduan Setup

Google Sheets
API Setup

Ikuti panduan langkah demi langkah ini untuk menghubungkan sistem laporan TERRAE TEA ke Google Sheets agar data tersimpan otomatis dan permanen.

⏱ Estimasi: 10–15 menit  ·  Level: Pemula
// Apa yang kamu butuhkan
Google Account
Akun Gmail biasa — tidak perlu akun berbayar.
Google Sheets
Spreadsheet baru sebagai database penjualan harian.
API Key
Kunci akses gratis dari Google Cloud. Dibuat sekali, dipakai selamanya.
Biaya
Google Sheets API gratis. Batas gratis 100 req/100 detik — lebih dari cukup untuk laporan harian.
// Langkah 01
STEP01
Google Cloud Console
Buat Google Cloud Project
Buka URL ini
Pergi ke Google Cloud Console:
URL https://console.cloud.google.com
A
Login dengan akun Google kamu
Gunakan Gmail yang sama dengan Google Sheets. Setujui Terms of Service jika diminta. Tidak perlu kartu kredit.
B
Klik "Select a project" di header
Klik dropdown di atas, lalu klik NEW PROJECT.
Header›Select a project›NEW PROJECT
C
Isi nama project dan klik CREATE
Contoh nama: terrae-tea-sales. Klik CREATE dan tunggu project selesai dibuat.
// Langkah 02
STEP02
APIs & Services
Aktifkan Google Sheets API
A
Buka menu APIs & Services → Library
Di sidebar kiri:
☰ Menu›APIs & Services›Library
B
Cari "Google Sheets API"
Ketik Google Sheets API di kolom pencarian, lalu klik hasilnya.
C
Klik tombol "ENABLE"
Klik ENABLE dan tunggu hingga status API aktif.
Konfirmasi
Kamu akan melihat tombol DISABLE dan statistik penggunaan — artinya API sudah aktif.
// Langkah 03
STEP03
Credentials
Buat API Key
A
Buka menu Credentials
Dari sidebar:
APIs & Services›Credentials
B
Klik "+ CREATE CREDENTIALS" → "API key"
API key akan langsung dibuat dan ditampilkan dalam popup.
C
Salin dan simpan API Key
// Contoh format API Key AIzaSyD1aBcDeFgHiJkLmNoPqRsTuVwXyZ12345678
Salin key dan klik CLOSE.
D
(Opsional) Batasi API Key
Klik nama key → API restrictions → Restrict key → centang Google Sheets API → SAVE.
Keamanan
Jangan bagikan API Key ke orang lain. Jika bocor, hapus dan buat key baru. Sistem ini menyimpan key di localStorage browsermu saja.
// Langkah 04
STEP04
Google Sheets
Siapkan Spreadsheet
A
Buat Spreadsheet baru
Buka sheets.google.com dan buat spreadsheet baru. Beri nama: TERRAE TEA — Laporan Penjualan.
URLhttps://sheets.google.com
B
Salin Spreadsheet ID dari URL
// Format URL Google Sheets https://docs.google.com/spreadsheets/d/SPREADSHEET_ID_ADA_DI_SINI/edit // Contoh ID nyata: 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms
Bagian di antara /d/ dan /edit adalah Spreadsheet ID-mu.
C
Pastikan nama tab adalah "Sheet1"
Klik kanan tab bawah → Rename → ketik Sheet1 → Enter.
// Langkah 05
STEP05
Sharing & Permissions
Atur Akses Spreadsheet
⚠️
Langkah Kritis
Ini langkah yang paling sering dilewati dan menyebabkan error. Spreadsheet harus dibagikan agar bisa diakses API.
A
Klik tombol "Share" di spreadsheet
Di pojok kanan atas Google Sheets, klik Share.
B
Ubah ke "Anyone with the link — Editor"
Di bagian General access → pilih Anyone with the link → role: Editor.
Share dialog›General access›Anyone with the link — Editor
C
Klik "Done"
Spreadsheet siap diakses via API.
// Langkah 06
STEP06
Sistem TERRAE TEA
Hubungkan ke Aplikasi
A
Buka tab "Laporan Penjualan"
Klik tab 📊 Laporan Penjualan di bagian atas halaman ini.
B
Masukkan Spreadsheet ID dan API Key
// Isi kolom config di tab Laporan Penjualan Spreadsheet ID : 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms API Key : AIzaSyD1aBcDeFgHiJkLmNoPqRsTuVwXyZ12345678
C
Klik "CONNECT" dan verifikasi
Jika berhasil, muncul notifikasi hijau: "✓ Terhubung ke: [nama sheet]". Konfigurasi tersimpan otomatis.
D
Input data → Klik "SIMPAN KE GOOGLE SHEETS"
Data akan masuk ke Sheet1 dengan kolom: Tanggal, Waktu, Produk, Harga, Stok, Terjual, Sisa, Profit.
Setup Selesai!
Sistem TERRAE TEA sudah terhubung. Setiap simpan akan otomatis terekam di spreadsheet dengan timestamp lengkap.

// FAQ & Troubleshooting
Troubleshooting
Solusi untuk masalah paling umum saat setup.
Error "403 Forbidden" saat CONNECT
API Key belum benar atau Google Sheets API belum diaktifkan. Selesaikan langkah 02 terlebih dahulu.
Error "400 Bad Request" saat SIMPAN
Nama tab sheet salah. Pastikan tab bernama persis Sheet1 (S kapital, tanpa spasi).
Error "The caller does not have permission"
Spreadsheet belum dishare. Kembali ke langkah 05, ubah General Access ke Anyone with the link — Editor.
CONNECT berhasil tapi SIMPAN gagal
Cek API Key restriction. Pastikan Spreadsheet ID tidak ada spasi ekstra di depan/belakang.
LOAD DATA tidak menampilkan apa-apa
LOAD DATA hanya memuat data hari ini. Simpan dulu datanya sebelum dicoba di-load.
API Key hilang setelah browser ditutup
Pastikan tidak menggunakan mode Incognito/Private. localStorage terhapus di mode private.
